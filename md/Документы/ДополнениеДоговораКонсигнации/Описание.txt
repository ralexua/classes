Документ "Дополнение к договору консигнации" содержит до-
полнительную информацию, которая используется при вычислении 
вознаграждения комиссионеру. Вознаграждение рассчитывается ис-
ходя из значений двух процентов: "От суммы" и "От прибыли". 

Вознаграждение "от суммы" - это процент от суммы реализации, 
который получает комиссионер при продаже товара по минимальным 
ценам (по ценам, по которым товар был передан комиссионеру). 
Вознаграждение "от прибыли" - это процент от разности между 
суммой, за которую был продан комиссионный товар и его учетной 
стоимостью, который получает комиссионер при продаже товара 
по ценам выше минимальных.

Документ "Дополнение договора консигнации" используется в 
качестве договора при проведении взаиморасчетов по вознагражде-
нию за услуги консигнации, полученному от комиссионера (выпла-
ченному консигнатору). Оплата услуг консигнации осуществляется 
в момент продажи комиссионного товара как взаимозачет между ко-
миссионером и консигнатором.

Группа флажков "Способ формирования налоговых документов" 
служит для задания режима формирования налоговых накладных 
по документам, оформляющим поставку услуг консигнации по данному 
договору.                                                                  

При проведении документ регистрирует себя в справочнике "До-
полнения договоров" в периодическом реквизите "Дополнение". 
По одному договору может быть оформлено несколько дополнений 
с различными ставками вознаграждения. Дополнение влияет на 
расчет вознаграждения в период начиная с даты документа по 
дату ввода другого дополнения такого же вида.
