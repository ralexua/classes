Счет, вообще говоря, не является обязательным документом при 
оформлении товародвижения. Он нужен для фиксации 
предварительной договоренности о приобретении контрагентом 
товара, и может являться документом, на основании которого 
производится оплата и отгрузка товара. Заказанный по счету товар 
при проведении счета резервируется (см. регистр "РезервыТоваров") 
компанией в целом. Резерв товара можно посмотреть в отчете 
"Резервы товаров".

Счета создаются, хранятся и редактируются в журнале "Счета" 
вместе с документами "Снятие резерва" (см. ниже). Новый счет 
вызывается также с помощью меню "Документы" / "Расходные" / " 
Счет" или с помощью кнопки на панели "Инструменты 
Администратора".

Резервирование товара автоматически снимается при отгрузке 
товара по данному счету (то есть, при проведении выписанной на 
основании данного счета расходной накладной или расходной 
розничной накладной). 

В программе предусмотрено и ручное снятие 
резерва по товарам, выписанным в счете, с помощью 
формирующегося на основании счета документа "Снятие резерва". 
Возможно также снятие резерва по конкретному товару, 
зарезервированному по счету.

На основании счета на оплату могут быть выписаны следующие 
подчиненные документы:
- "Приходный кассовый ордер" - для фиксации наличной 
оплаты по счету. Безналичная оплата по счету фиксируется с 
помощью документа "Движения денежных средств". 
- "Расходная накладная" или "расходная розничная" для 
регистрации отгрузки товаров по счету;
- "Акт снятия резерва" для ручного снятия резерва товара по 
счету. 

Счет не выписывается на основании какого-либо документа.

Начального заполнения состава счета нет.
