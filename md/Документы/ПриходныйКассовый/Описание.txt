Документ "Приходный кассовый ордер " (ПКО) предназначен для 
учета поступлений наличных денежных средств от покупателей 
в качестве расчетов за отгруженные товары, реализованные 
товары в случае оформления операции передачи на реализацию, 
и предоставленные услуги. 

ПКО создаются, хранятся и редактируются в журнале "Кассовые" 
вместе с РКО. Новый ПКО вызывается также с помощью меню 
"Документы" / "Кассовые" / "Приходный кассовый" или с помощью 
кнопки на панели "Инструменты Администратора". 

При начальном заполнении сумма приходного кассового ордера 
заполняется суммой долга покупателя, рассчитанного по всем 
произведенным с ним операциям. Сумма долга рассчитывается с 
учетом того, каким образом был выписан документ. 

- Если ПКО выписан на основании расходной накладной, его 
сумма заполняется суммой неоплаченных товаров по данной 
накладной.

- Если ПКО выписан на основании счета, его сумма заполняется 
суммой отгруженных, но не оплаченных товаров по данному 
счету. 

- Если ПКО выписан в рамках договора, его сумма заполняется 
суммой отгруженных и/или реализованных, но не оплаченных 
товаров по данному договору. 

- Если ПКО выписывается для контрагента безотносительно 
договора и без привязки к конкретному документу, его сумма 
заполняется суммой отгруженных и/или реализованных, но не 
оплаченных товаров. В расчет принимаются только документы, 
которые не принадлежат ни одному из договоров данного 
контрагента.

Если для приходного кассового ордера установлен 
управленческий или общий тип учета, то при проведении документ 
производит изменения во взаиморасчетах с контрагентами. Если есть 
задолженность контрагента, то погашается задолженность, начиная с 
самых ранних неоплаченных накладных или документов, 
оформляющих факт реализации товара (продажа реализатора)
(если только ордер не выписан по конкретному документу - 
в этом случае погашаются задолженности именно по этому
документу). При этом учитывается договор, к которому 
принадлежат данные документы, погашаться будут суммы 
только в рамках одного и того же договора. 

В зависимости от состояния взаиморасчетов с покупателем, 
внесенная сумма может быть при проведении документа 
автоматически разбита (при необходимости) на отдельные части: 

- оплата отгруженных товаров (оказанных услуг);

- оплата реализованных товаров;

- поступление аванса от покупателя. 

Каждая из этих частей внесенной суммы отражается в 
соответствии с правилами бухгалтерского учета.

Для документов, относящихся только к управленческому учету 
(тип учета "Упр."), бухгалтерские проводки экспортироваться не 
будут.
