Документ "Грузово-таможенная декларация" ("ГТД") предназна-
чен для регистрации факта получения товаров от иностранного по-
ставщика.

ГТД создаются, хранятся и редактируются в журнале "ГТД". Но-
вый документ вызывается также с помощью меню "Документы" / "Приходные" / "ГТД".
В качестве клиента в документ "ГТД" должен быть выбран 
контрагент с установленным видом контрагента "Нерезидент".

В данном документе присутствуют дополнительные поля, а именно:

		В группе реквизитов "Таможня".                                                 
	Таможня - поле для выбора контрагента - таможни из справоч-
ника "Контрагенты". В данное поле должен быть выбран контрагент 
с видом "Обязательные платежи".
	Договор - поле для выбора договора взаиморасчетов с тамож-
ней.
	Кред. док. - документ взаиморасчетов с таможней.
	
		В группе реквизитов "Перевозка".
	Перевозчик - поле для выбора контрагента - перевозчика из 
справочника "Контрагенты".
	Договор - поле для выбора договора взаиморасчетов с перевоз-	
чиком.
	Кред. док. - документ взаиморасчетов с перевозчиком.

    Табличная часть документа расположена на закладке "ГТД". 
Там же располагаются следующие дополнительные поля.
	Пошлина - поле для ввода суммы пошлины в базовой 
валюте, без учета налога на добавленную стоимость.
	Услуги таможни - поле для ввода суммы таможенных 
услуг в базовой валюте, без учета налога на добавленную стоимость.
	Акциз - поле для ввода суммы акциза в базовой валюте, без уче-
та налога на добавленную стоимость.

		В группе реквизитов "Перевозка"
	Валюта - поле для выбора валюты перевозки.
	Курс - поле для ввода курса валюты перевозки.
	Дата - поле для ввода даты курса валюты перевозки.
	Сумма - поле для ввода суммы перевозки в выбранной валюте.
       
При проведении документа суммы пошлины, акциза, таможенных 
услуг и перевозки будут включены в себестоимость приходуемых товаров.

Возврат товаров, оприходованных по ГТД, не предусмотрен.
На основании ГТД можно выписать следующие подчиненные документы:
"Расходный кассовый ордер" - для наличной оплаты полученного товара. 
"Запись книги приобретения" - для того, чтобы зарегистрировать 
запись книги приобретения.
