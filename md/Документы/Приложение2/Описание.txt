Для регистрации изменений количественных и стоимостных по-
казателей выписанных покупателю налоговых накладных, а также 
для формирования печатной формы соответствующего документа 
(«Расчет корректировки») в типовой конфигурации предусмотрен 
документ «Приложение №2».  

Документ может быть введен только на основании документа 
«Налоговая накладная». Реквизиты шапки полностью заполняются 
из документа – основания.    

Значения реквизитов «ТМЦ», «Ед», «Кво», «Цена -» табличной 
части документа заполняются из документа-основания и недоступны 
для редактирования. В остальных реквизитах табличной части ука-
зывается следующая информация:
- причина корректировки, необходимая при формировании 
печатной формы документа;
- изменение количества по выбранному товару;
- изменение цены без НДС по выбранному товару;
- изменение суммы НДС;
- изменение суммы без НДС (базы налогообложения).   

Отметим, в соответствии с действующим законодательством, 
корректировку по товару можно проводить либо по цене, либо по 
количеству.         

При проведении документ не формирует бухгалтерских проводок, 
но для того, чтобы документ попал в отчет «Книга продаж», он дол-
жен быть проведен.
